# 小程序图标资源说明

本目录需要放置以下图标文件：

## TabBar 图标 (建议尺寸: 81x81 像素)

1. `device.png` - 设备图标（未选中）
2. `device-active.png` - 设备图标（选中）
3. `focus.png` - 专注图标（未选中）
4. `focus-active.png` - 专注图标（选中）
5. `stats.png` - 统计图标（未选中）
6. `stats-active.png` - 统计图标（选中）

## 其他图标

7. `clock-icon.png` - 时钟设备图标 (100x100)
8. `astronaut.png` - 太空人图片 (200x200)
9. `empty-device.png` - 空设备状态图 (300x300)

## 图标获取建议

可以从以下网站获取免费图标：
- https://www.iconfont.cn/
- https://icons8.com/
- https://www.flaticon.com/

或者使用 Emoji 作为临时替代（代码中已有 Emoji 备选方案）

## 临时解决方案

如果暂时没有图标，小程序仍可运行，但 TabBar 会显示为空白图标。
建议先使用纯色方块图片作为占位。
